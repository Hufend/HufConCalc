### 用法

由于作者的编程水平低下，所以运算符号的后边一定要加空格  
例如：

	8* 2+ ( 9- 3) / 2+ 1=
	20.000000

为了美观一般会写成以下形式

	8 * 2 + ( 9 - 3 ) / 2 + 1 =
	20.000000

当然还有几种**特别**的写法  

因为我~~局域网~~懒  
所以程序中的算式可以是等号前无限换行、大中小括号乱用  
例如：

	8
	*
	2 + (
	9 -
	3 ) /
	2
	+ 1 =
	20.000000

	9 * ( 8 + { 6 * { 7 * 9 - 2 ) + [ 6 + 3 * 3 ) - 81 ] - 306 } =
	18.000000
